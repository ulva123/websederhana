VISI

Program Studi Diploma Tiga Teknik Informatika 2016-2020: Menjadi program studi vokasi yang menghasilkan lulusan bidang rekayasa perangkat lunak berstandar Nasional dan menuju reputasi Internasional


Misi

Program Studi Sarjana Terapan Rekayasa Perangkat Lunak adalah:

  1. Pendidikan: Mendidik dan Meningkatkan kemampuan mahasiswa dalam menerapkan Ilmu Pengetahuan Dan Teknologi Rekayasa Perangkat Lunak Bidang Rekayasa Web Dan Aplikasi mobile untuk meningkatkan daya saing tinggi
  2. Penelitian: Menciptakan, mengambangkan dan menerapkan ilmu pengetahuan dan teknologi untuk menghasilkan produk unggulan rekayasa perangkat lunak
  3. Pengabdian: Meningkatkan pemanfaatan teknologi informasi dalam mengatasi permasalahan-permasalahan di lingkungan internal maupun lingkungan eksternal(industri, pemerintah dan masyarakat umum).
